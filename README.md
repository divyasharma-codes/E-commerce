# E-Commerce App - React Native

A fully functional E-commerce mobile app built with React Native + Expo. Includes product listing, cart management with empty state handling, and responsive UI.

## ✨ Key Features

| Feature | Description |
| --- | --- |
| **Product Catalog** | Browse products in grid layout |
| **Cart Management** | Add/Remove items with quantity update |
| **Empty Cart State** | Shows "No product have been added" when cart is empty |
| **State Management** | Redux Toolkit for global state |
| **Cross Platform** | Runs on Android & iOS via Expo |

## 🛠 Tech Stack

- **Framework**: React Native 0.74 + Expo SDK 51
- **State**: Redux Toolkit
- **Navigation**: React Navigation
- **Icons**: @expo/vector-icons
- **Language**: JavaScript

## 🚀 How to Run

```bash
# 1. Clone repo
git clone https://github.com/divyasharma-codes/E-commerce.git

# 2. Install packages

# 3. Start app
npx expo start
